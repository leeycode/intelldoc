# 尽调报告撰写
# 项目介绍

通过填充模板，生成报告，模板有excel表格或者word。规划下面功能。

## 技术方案
前端使用vue2.0,ant-design-vue 1.7.2,webpack5.0,不要使用vuex，数据也写在页面vue data中。

## 模板管理
包含下面内容：
1. 模板列表
2. 模板编辑，
    可以编辑如下信息
    报告分类：尽调/公文/研报/新闻稿
    基本信息：报告名称，客户名称，统一社会信用证，描述，作者，写作日期等等，作为复选框，用户可以选择一个或者多个。
    模板类型：用户上传的word模板，excel模板，或者excel模板，模板中需要填充的部分用{{客户信息.cust_name}}标识，"客户信息"是数据源主题，"cust_name"是数据源字段。
3. 模板删除

## 报告管理
1. 选择模板
2. 填写基本信息
3. 点击生成报告
4. 在线预览
    生成完成报告，可以在线预览docx文件,excel文件,也可以导出
5. 报告审核
    通过大模型接口对内容进行审核
6.保存导出
7.报告列表

## 素材管理
素材列表
    用户可上传图片，文档等素材，也可以查看公共素材

## 数据源管理
列表包含主题，接口名称，url,数据来源，查询详情等
接口分为两类，一类是系统的http接口，一类是agent接口。
点击查询详情，展示数据源详情，可以编辑，删除。
